## Install
=========
git clone in to ~/.config/

## Install VIM-Plug
=========

```
curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

## Install Plugins
=========

get into nvim
run :PlugInstall

## Update Config
=========
```
nvim ~/.config/nvim/init.vim
```
